# RoboMagic-PathToX-AlSiddiq-Robotics
AlSiddiq Robotics: Path to X Competition  A robotics competition where robots navigate a path autonomously using line-following, then switch to remote control to collect boxes labeled X and O, which are later used to compete against another robot in the XY game
AlSiddiq Robotics 2026: Path to X Competition

Our robot was designed with two operational modes: autonomous line-following and remote-controlled operation.

Software: Arduino IDE , solidworks and kicad.

Workflow: The project began after the competition announcement, starting with multiple meetings to finalize the mechanical design of the robot body and arm. We then selected the components and motors, leaving the software development for last.

Challenges: Mechanical and hardware development took longer than expected, limiting the time for software testing and integration.

Strengths: Strong mechanical design, high-quality motors, and a smart, cost-effective gripper.

Weaknesses: The robot is relatively simple and heavy; achieving the desired design sometimes required bending the rules.
